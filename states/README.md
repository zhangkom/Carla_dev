# Local Carla State Files

Save GUI-authored `.carxs` plugin states here during Windows debugging.

These files are intentionally ignored by Git because they may contain local plugin paths, preset metadata, or commercial plugin state. Keep only stable names in config, for example:

```text
states\kong_qinrv_gaohu_sus_leg_mw.carxs
```
